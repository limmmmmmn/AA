# Appears! Appears! Appears!
### A AAA Incremental JRPG — *"A whole JRPG. Two rooms."* (GDD v2.0 기반)

## 실행 방법
1. Godot 4.5 이상 실행 (`다운로드` 폴더의 `Godot_v4.5.1-stable_win64.exe`)
2. 프로젝트 목록에서 이 폴더(`C:\Appears! Appears! Appears!`)를 임포트/열기
3. **F5** (메인 씬 실행) · **F11** 전체화면

## 조작 — 양손 분업 (키보드 = 용사의 몸, 마우스 = 시선)
| 입력 | 동작 |
|---|---|
| **WASD / 방향키** | 파티 선두 이동 (꼬리물기 대열) |
| **Space** | 상호작용 — 항아리·건물·NPC·성문·이정표·몬스터 |
| **Tab** | 마을 재건 계획도 열기/닫기 |
| 마우스 | 시선 전용 — 전투창 호버(주시 버프), 황금 슬라임 문지르기, 도크, 계획도, 미니게임 |
| ESC | 메뉴/계획도 닫기 |

- 커서 3종: 포인터(기본) / 눈(전투창) / 쓰다듬는 손(황금 슬라임·대장간)
- 카메라 없음 — **룸 = 정확히 한 화면 (640×360)**, 룸 전환은 즉시 페이드

## 월드 구조 — 기지 ↔ 필드
- **기지** 1개: 여관·촌장(계획도)·수배 게시판은 처음부터. 나머지 건물은 계획도로 건설. 오른쪽 끝 = **성문** (Space → 행선지 선택).
- **필드 5개** (좌 = 출구, 우로 갈수록 위험, 최심부 = 지배자):
  - **초원**: 기본 · **숲**: 발굴 반짝이 풍부 · **동굴**: 무리 조우 · **설원**: 정예 단일 + 황금 슬라임 잦음 · **마왕성**: 서사시 완독 후 입장
- **전투창은 룸 전환을 관통한다** — 창 잔뜩 열고 기지로 돌아와 마을일 보기 = 의도된 테크닉. 상단 밴드 도킹, 4개 초과 시 창이 한 단계 작아짐 (하단 1/3 보호 구역).
- **귀향 연출**: 부재중 조수들(원숭이/상자지기)이 모은 골드를 귀환 순간 한꺼번에 수확. 꽃돼지는 일행과 동행하며 필드 발굴.

## 마을 재건 계획도 (Tab)
촌장이 관리하는 노드 트리 — 상점을 대체. 열려 있어도 게임은 계속 돈다.
- **전투 가지**: 전투창 확장 / 가속 / 공격·HP·골드 그리드
- **원정 가지**: 이속 / 삽 / 용사의 직감 / 행동반경 / **필드 해금 대형 노드**
- **마을 가지**: 대장간·교회·보물상자·음유시인·카지노 + 항아리 확충
- **조수 가지**: 원숭이 / 상자지기 / 꽃돼지
- **이중 열쇠**: 마일스톤 달성 → "소문을 들었다: …" 와 함께 노드 출현 → 골드 지불로 구매. 건물 구매 → NPC가 걸어 들어와 건설.
- 마우스 드래그로 팬. 오른쪽 **도크**가 점등하면 = 볼 일이 생겼다는 신호.

## 시스템 요약
- 필드별 **수배 게시판 탭**: 수배서 3장 → 그 필드의 지배자가 깨어난다 → 처치 시 훈장 확정 + 다음 필드 소문
- 유령/교회/전멸, 대장간 리듬 미니게임(회심의 필살작), 카지노 슬롯+교환소, 서사시 6절(전설의 검 성장 + 마왕성 열쇠), 훈장 10종(슬롯 3+, 영구 소장), 황금 슬라임 문지르기, 용사의 직감(사냥→귀환→재출격 완전 자율 루프)
- 원격(시선) 가능: 계획도/카지노/대장간/서사시/수배 — **원격 불가(몸)**: 여관·교회·항아리·발굴·보스
- 마왕 처치 → 엔딩 → 모험의 서(회차)
- 자동 저장. 구버전(v1) 세이브는 구조가 달라 초기화됨.

## 임시 에셋 (나중에 png/사운드 교체)
- 커서 3종(`ui_lib.gd`), 조수 3종 8×8(`assistant.gd`), 게시판/이정표/성문 표식/반짝이/말풍선/카지노 전구/대장간 불씨(도형), 성문=castle.png, 카지노=상점 보라 틴트, 음유시인=촌장 초록 틴트, 효과음 전부 칩튠 합성(`sfx.gd`)

## 개발용
- `AAA_SMOKE=1` → 자동 테스트 (계획도/출격/전투/지배자/귀향 수확/건설/카지노/서사시/전멸/세이브)
- `AAA_SMOKE=1` + `AAA_SHOT=1` → 스크린샷 5장 · `AAA_FONT=1` → 폰트 테스트
- 세이브: `%APPDATA%\Godot\app_userdata\Appears! Appears! Appears!\appears_save.json`
