# Appears! Appears! Appears!
### A AAA Incremental JRPG — *"A whole JRPG. One screen."* (GDD v3.1 기반)

## 실행 방법
1. Godot 4.5 이상 실행 (`다운로드` 폴더의 `Godot_v4.5.1-stable_win64.exe`)
2. 프로젝트 목록에서 이 폴더를 임포트/열기 → **F5** · F11 전체화면

## 대원칙 (v3.0)
> **모든 해금은 세계의 변화로만 표현된다. 해금 전용 UI는 존재하지 않는다.**
> 노드버스터류는 그래프를 키우고, 우리는 **마을을 키운다.**

## 조작
| 입력 | 동작 |
|---|---|
| **WASD / 방향키** | 이동 |
| **Space (몸)** | 인접 상호작용 — 여관/교회/항아리/발굴/이정표/커맨드 메뉴 |
| **클릭 (시선)** | 원격 — 대장간 불 켜지면 클릭→리듬 미니게임, 카지노 클릭→슬롯 |
| 전투창 호버 | 주시 버프 · 황금 슬라임 드래그 = 문지르기 |
| ESC | 메뉴 닫기 |

## 한 화면 월드 (640×360, 카메라 없음)
```
[마을 ⅓]           [필드 ⅔]                [지배자]
 촌장·주민·항아리    몬스터·발굴 (좌→우 위험)   결계+실루엣 상주
 잠긴 창고·붉은상자  (이정표: 필드 스왑)
```
- 필드 5종(초원/숲/동굴/설원/마왕성)은 **이정표**(Space)로 우⅔만 통째 교체. 다음 필드는 지배자를 쓰러뜨리면 열린다.
- 지배자는 시작부터 **결계 속 실루엣**으로 보인다. 수배서 3장 → 결계 해제 → 보스전 = 파티 정지, 대형 창 하나, 호버 올인 (집중 게임).

## 주민 = 진행바 (부흥 단계)
- 시작: 용사·촌장·항아리 2개·**잠긴 창고·붉은 상자**뿐. HUD도 없다 (전부 이벤트로 탄생).
- 촌장 Space → 설명창 탄생 → 첫 골드 → 카운터 탄생 → 첫 전투창 → HP창 탄생 → 골드 50 → **촌장의 부탁 개방**.
- **부탁 메뉴에 영입 후보 2~3명 동시 노출** — 어느 부탁부터가 곧 선택:
  여관 주인(슬라임 5) → 대장장이(250G·Lv3) → 신부(박쥐 6) → 상인(400G) → 음유시인(보스) → 메달왕(메달 3) → 도박사(1500G·Lv6)
- 조건이 뭐든 결과는 하나: **사람이 마을에 걸어 들어오고 시설이 선다.**
- 주민 **3명 → 광장 확장** / **6명 → 분수+성벽**. 마을에 서 있는 사람 수가 곧 진행바.

## 해금 문법 6종 (노드 트리 폐지)
① 골드→**건물별 커맨드 메뉴** (촌장: 부탁·건설·전선·원정 / 여관: 침구 / 대장간: 무기 / 상점: 골드·조수 / 교회: 부활·모험의 서 / 메달왕: 훈장 교환)
② **레벨 게이트** (수배 게시판 Lv2, 대장장이 Lv3, 직감 Lv5, 도박사 Lv6…)
③ **주민 영입 = 시설 화신** ④ **부탁 퀘스트** ("슬라임 5마리를…")
⑤ **드퀘 열쇠** — 잠긴 창고(도둑의 열쇠=초원 보스), 붉은 상자(마법의 열쇠=동굴 보스). 시작부터 보인다.
⑥ **작은 메달** — 항아리 3%/상자 15%/발굴 10% 드랍 → 메달왕에게 훈장 교환

## v3.1 — 콘텐츠 대확장
- **동료 18명** — 정규 7(시그니처 무기, 대장간 강화) + 객원 11(1줄 패시브). 여관에서 **파티 편성**(최대 5, 용사 고정), **《동료들의 서》**는 회차를 넘어 남는다.
  - 영입 경로: 부의 소문(기사60/마법사350/사제1000/전사2000/무도가4000 누적골드) · 주민 연동(여관→요리사, 상점→상인, 음유시인→바드, 신부→스님) · 촌장 부탁(사냥꾼/무희/광부/어부형제/은행원) · 서사시(드루이드/도적)
- **합체기 딱 3종** — 편성 퍼즐: 참치 어택(어부 형제)·개구리의 왈츠(드루이드+바드)·명계의 행진(사제+유령). 승리로 게이지 충전 → 파티 발광 → **클릭으로 발동** (컷인+필드 스윕).
- **서사시 = 사건 판매** — 절마다 세계가 변한다: 드루이드 등장 / 검이 꽂힌 바위(Lv10에 뽑기) / 도적 합류 / **도적의 배신과 귀환**(훈장 「의리의 심장」) / 합체기 힌트 / 마왕성의 문.
- **호버 대확장** — 주시 창: 금가루+루프음+테두리 맥동. 교회 "축복": 주시 강화/치유의 눈길(멤버 박스 호버 힐)/황금의 손길(황금 슬라임 포획 해금)/성수 그릇/샘의 축복. **성수 게이지**(자동 재생, 힐로 소모). 퇴각 나팔 = 호버 중 도망 버튼.
- **은행** — 은행원 영입 시 건물+개구리 석상(Space=잔돈 예금). 예금은 **전멸에도 불가침**, 접속 중 30초마다 이자.
- **상인의 텔레파시** — 직감 이후 해금. 건물 클릭=원격 메뉴 (쉬기/단조/슬롯/입출금 등 몸 행위는 잠김).
- XP 미니바(정보창)+파란 경험치 입자+레벨업 의식(플래시+게이트 힌트). 침구 개선·사제 기도는 **%화**.

## 임시 에셋 (png 교체 예정)
커서 3종·조수 3종(코드 생성), 게시판/이정표/창고/자물쇠/분수/결계/왕관/은행/개구리 석상/검 바위(도형), 객원 동료(촌장 스프라이트+틴트), 합체기 컷인(`assets/combined/{tuna,frog,skeleton}.png` 있으면 사용, 없으면 기존 몬스터+틴트), 효과음 전부 합성(`sfx.gd`)

## 개발용
- `AAA_SMOKE=1` → 자동 테스트 (공개 스케줄/주민/부흥/보스/열쇠/메달왕/스왑/편성/합체기/축복/은행/전멸/세이브v5)
- `AAA_SMOKE=1`+`AAA_SHOT=1` → 스크린샷 3장 · 세이브: `%APPDATA%\Godot\app_userdata\Appears! Appears! Appears!\`
- 테스트 후 세이브 삭제 필수 (다음 런 오염 방지)
