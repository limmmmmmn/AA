## v4.0 (GDD v3.8) — 마을 중심 재편: "건물이 곧 진행바, 기물이 곧 잔돈의 기쁨"
- **"!" 마커 시스템**: 마을 오브젝트·건물·NPC 머리 위 **금색 느낌표** (통통 튀는 공용 마커 1종) = *지금 실행 가능한 일이 있다*. 골드가 모자라면 안 뜨고, 골드가 차오르면 마을 곳곳에 느낌표가 켜진다 — "쇼핑 갈 시간"의 디제틱 신호. 1초 폴링(`can_shop`)으로 일괄 갱신.
- **촌장 분업**: 촌장 = 부탁(조건형 주민) + **건설 카탈로그** + 칭호 대사만. 수치 업글은 이사 — **훈련소(신규)**: 전투창 확장/전투 가속/무리 유인/퇴각 나팔. **마구간(신규)**: 이속/직감/행동반경/탈것. **상점**: 골드 감각 + 도구(삽·곡괭이·등불·텔레파시) + 조수. 촌장 메뉴는 이제 스크롤 없이 한 화면.
- **건설 카탈로그 (건물/기물 2부)**: 건물 = 훈련소 200G/대장간 250/마구간 300/무기점 350/상점 400/카지노 1500/은행 2500 — 구매하면 **건물이 먼저 팝 → 주인이 걸어 들어와 입주** ("건물이 서면 사람이 온다", 부탁 순서의 반전). 기물 = 가로등 40G(밤에 점등+글로우)/허수아비 50(두드리면 잔돈, 45s 쿨타임)/게시판 60/우물 80/개구리 석상 120(은행 후) + 항아리 확충 — 값싸고 즉각적인 "잔돈 쇼핑" 층.
- **부흥 단계 = 건설물 수** (4/8/12/15): 인원 카운트 폐지 — 마을이 "사람이 늘었다"가 아니라 **"마을이 자랐다"**로 읽힌다. `Game.built_count()` = 건물+기물+항아리.
- **상단 파티 카드 (하드코딩 규격)**: 통합 바 폐지 → **개별 DQPanel 4장** — 정보 캡슐과 동일 높이(28px)·동일 y(8px), 폭 88 균일, 간격 8, 컨테이너 자동 흐름 금지(절대좌표 상수 — Container 금지 규칙의 명시적 예외). 오의 게이지·성수는 카드 열 우측 끝 소형 박스.
- 전투창 몬스터 = **원본 1배수 고정** (필드와 같은 크기). 세이브 마이그레이션: 게시판→기물, 은행 세이브→개구리 석상 자동.

## v3.9 (GDD v3.7) — 구조 다이어트: "8인의 동료, 1개의 오의, 전장이 된 필드"
- **동료 8인 고정 / 파티 4인**: 용사·기사·전사·사제·마법사·도적·무도가·드루이드 — 전원 정규 동료(무기 슬롯 보유). 나머지 옛 "객원"들은 주민으로 재분리 (은행원·어부 형제 = 주민 부탁). 세이브 마이그레이션이 구 동료를 자동 정리한다.
- **오의 = 장착식 (FF6 마석 문법)**: 편성 조건 폐지 — **오의서**를 찾아 여관에서 1개 슬롯에 장착. 획득처 3종: 참치 어택 = 수중 필드 최심부의 오의서 / 개구리의 왈츠 = 개구리 저금통 누적 500G / 명계의 행진 = 동굴 지배자 격파. 첫 오의서는 자동 장착, 게이지는 장착 중일 때만 찬다.
- **필드 = 전장**: 전투창이 **인카운트 지점 근방에 스폰** (어디서 싸움이 났는지가 공간으로 읽힌다) + 겹침은 AABB 밀어내기로 해소 (피격 흔들림 복귀 지점도 함께 이동). 겹쳐보기 스택은 **소집 토글** 버튼(소집/해산)으로 임의 전환.
- **HUD 상단 스트립**: 파티 카드 4칸 가로 1열 [카드×4 | 오의 게이지 | 성수] — 상단 정보 캡슐 오른쪽에 나란히. 게이지/성수 없으면 칸막이째 숨김.
- **오프닝 확정**: 타이틀 배경 = **항상 빈 초원** (신규든 이어하기든) + PRESS **ANY** KEY → 로고 상단 축소 이동 + 메뉴 → 시작하면 저장된 마을이 **뾱! 뾱! 뾱! 팝인으로 재등장** — "Appears!"가 매 부팅의 의식. 새벽빛은 그라디언트.
- 앰비언트 팝 삭제 → 희귀 이벤트 박스 대사로 통합. 식사 개선(여관)/진혼곡·여운(교회)/곡괭이(촌장)/추적(게시판) 업글 신설.

- **통합 파티 바**: 하단 = 단일 패널 [합체기 게이지 | 파티 칸들(이름+HP바, 숫자는 호버 툴팁) | 성수]. 칸막이 1px 크림, 유령 칸 = 청색 반투명.
- **텍스트 3계층**: 상시 설명창 폐지 → ① 툴팁(커서 추종 — 검시/호버) ② 토스트(우상단) ③ **이벤트 박스**(중앙 하단 + 화면 비네트 + 큐 + 촌장/엄마 초상화 슬롯).
- **앰비언트 팝**: "바람이 분다. 평화롭다." → 0.6~1초 뒤 비껴난 자리에 "…일단은." — 수 분당 1회, 어긋남이 곧 뉘앙스.
- **텍스트 마크업**: [shake]/[wave](내장) + [slam]/[whisper](커스텀 이펙트). 수중 필드의 선언은 전부 출렁이고, 참치 컷인은 "[slam]…참치다.[/slam]"가 쿵 하고 박힌다. 컷인은 비스듬히(연기 레이어 — 가구는 그리드, 연기는 자유).
- **신규 보스 「복수(複數)의 감시자」** (설원): 본체 없음 — **기울어진 전투창 6개가 곧 보스**. 제한 시간 내 전부 처치 못 하면 감긴 눈이 다시 뜬다. 호버 배분(시선 경제)의 시험. 검시: "감시자가 당신의 시선을 불쾌해한다."

## v3.4 — 가독성·리듬 패치 (설계가 보이게 만드는 라운드)
- **전투창 120×90 (4:3)**: 몬스터 존 상단 2/3(크게) + 로그 1~2줄. **적 미니 HP바**(12×2px) + 공격 주체 스텝 + 5개씩 두 줄 도킹.
- **겹쳐보기 (교회 축복)**: 창들이 카스케이드 스택으로 정리 — ×N 뱃지, **스택 호버 = 전체 주시**, 클릭 = 셔플. 후반 "정신없음"이 업그레이드로 해소.
- **커맨드 창 좌측 고정** (동시 1개), 수배 게시판 = 현재 필드 탭 자동.
- **무리 스케일 = 편성 연동**: 창내 최대 적 = 파티 인원 (+무리 유인 = +1 보정). 동료 영입이 즉각 체감.
- **밤 시야 (등불)**: 밤엔 용사 중심 원형 시야만 밝다 (마을은 자유 지대). 촌장 "등불" 업글로 반경 확장, 최종 = 빛의 뱀.
- **무기점 / 대장간 이원화**: 무기점(신규 건물+무기상, 로스터 19) = 골드→플랫 공격력, **"공격력 37→41" 변화량 표시**. 대장간 = 벼림 % 배율(별개 슬롯) — 후반일수록 강한 선택적 손맛.
- **필드 확정**: 초원→숲→동굴→**설원(T4)**→마왕성. **수중 = 숨겨진 필드** — 어부 형제 부탁 완결 → 「바다의 노래」 → 이정표에 "≈ 수중" 등장 (물고기화·진주·어부 각성 = 발견한 자의 보상).
- **합체기 재편**: 명계의 행진 = 용사+마법사 / 개구리의 왈츠 유지 / **참치 어택 = 고전 4인 (용사+사제+마법사+도적)** — "전설의 조합이 완성되었다! …참치다." 파티창 옆 소형 게이지 상시 + 만충 점멸 + 토스트.
- **루팅 토스트 + NEW 뱃지**: 우상단 미니 창 슬라이드(훈장/메달 구분음), 도감에 [NEW] 표시(열람 해제). 중요 아이템은 중앙 팡파레 유지.
- **스탯 패널**: 파티창 클릭 → 이름/칭호/Lv/HP/공격력(플랫×벼림%)/무기/장착 훈장/작전.
- 기본 이름 "용사", "늦잠꾸러기"는 칭호로. 자율 이동은 필드 한정(마을 = 수동).

## 한 화면 월드 (640×360, 카메라 없음)
```
[마을 ⅓]           [필드 ⅔]                [지배자]
 촌장·주민·항아리    몬스터·발굴 (좌→우 위험)   결계+실루엣 상주
 잠긴 창고·붉은상자  (이정표: 필드 스왑)
```
- 필드 5종(초원/숲/동굴/설원/마왕성)은 **이정표**(Space)로 우⅔만 통째 교체. 다음 필드는 지배자를 쓰러뜨리면 열린다.
- 지배자는 시작부터 **결계 속 실루엣**으로 보인다. 수배서 3장 → 결계 해제 → 보스전 = 파티 정지, 대형 창 하나, 호버 올인 (집중 게임).

## 주민 = 진행바 (부흥 단계)
- 시작: 용사·촌장·항아리 2개·**잠긴 창고·붉은 상자**뿐. HUD도 없다 (전부 이벤트로 탄생).
- 촌장 Space → 설명창 탄생 → 첫 골드 → 카운터 탄생 → 첫 전투창 → HP창 탄생 → 골드 50 → **촌장의 부탁 개방**.
- **부탁 메뉴에 영입 후보 2~3명 동시 노출** — 어느 부탁부터가 곧 선택:
  여관 주인(슬라임 5) → 대장장이(250G·Lv3) → 신부(박쥐 6) → 상인(400G) → 음유시인(보스) → 메달왕(메달 3) → 도박사(1500G·Lv6)
- 조건이 뭐든 결과는 하나: **사람이 마을에 걸어 들어오고 시설이 선다.**
- 주민 **3명 → 광장 확장** / **6명 → 분수+성벽**. 마을에 서 있는 사람 수가 곧 진행바.
