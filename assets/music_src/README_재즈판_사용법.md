## 임포트 순서
1. FamiStudio 실행 → **File > Open** (또는 드래그&드롭)으로 txt 선택
2. **버전 주의**: 파일 첫 줄이 `Version="4.4.0"` — 텍스트 포맷은 **메이저.마이너가 일치해야** 읽힌다.
   설치한 FamiStudio가 4.3.x라면 첫 줄을 `Version="4.3.0"`으로 고치면 됨 (그 외 수정 불필요).
3. 좌상단 곡 선택 드롭다운에서 M1부터 재생.

## 시청 포인트 (명세서 §6 검증 게이트)
- ② 어깨가 살짝 들썩이는가 (2:1 셔플)
- ③ 게임 팡파레(C장조 상행)를 상상하며 겹쳐도 안 무너지는가 — G7♯5 두 박(B파트 8마디째 후반)은 컴핑을 비워 뒀다
- ①은 WAV 익스포트 후 1시간 재생 테스트

## 피드백 → 수정 루프
"A파트 더 스윙" / "멜로디 더 쉬게" / "베이스 더 걷게" 식으로 말해 주면
클로드 코드가 텍스트를 수정해 재출력. **확정되면** FamiStudio에서
**Export > WAV** (루프 포인트: 곡 시작 = LoopPoint 0) → `assets/`에 넣고
`Music._register`에 WAV 대입하면 인게임 교체 끝.
