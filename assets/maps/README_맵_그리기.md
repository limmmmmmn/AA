# 내 손으로 맵 그리기 (v4.3)

이 폴더에 PNG를 넣으면 **절차 생성 배경 대신** 당신이 그린 그림이 깔린다.
파일이 없으면 기존 자동 생성(잔디+나무)이 그대로 쓰인다. 즉, 그리고 싶은 것만 그리면 된다.

## 규격 (권장 크기 — 다른 크기여도 자동으로 늘려 채운다)

| 파일 이름 | 크기(px) | 무엇 |
|---|---|---|
| `village.png` | **216 × 360** | 마을 (화면 왼쪽 ⅓) |
| `field_0.png` | **424 × 360** | 초원 |
| `field_1.png` | 424 × 360 | 숲 |
| `field_2.png` | 424 × 360 | 동굴 |
| `field_3.png` | 424 × 360 | 설원 |
| `field_4.png` | 424 × 360 | 마왕성 |
| `field_5.png` | 424 × 360 | 수중 (숨겨진 필드) |

## 규칙
- **배경(땅·나무·바위·건물 외벽 등)만** 그리면 된다. 몬스터·항아리·건물 문·NPC·전투창은 게임이 그 위에 얹는다.
- 필드 그림의 좌표계: 그림 왼쪽 위(0,0)가 화면상 필드 영역의 왼쪽 위(마을 오른쪽 경계)에 붙는다.
- 픽셀아트 톤이면 도트가 안 뭉개지게 원본 배수(1×, 2×)로 그리는 걸 권장. 아무 크기나 넣어도 늘려서 채우긴 한다.
- 투명(알파) 사용 가능 — 안 그린 부분은 뒤가 비친다(현재는 뒤가 없으니 검게 보일 수 있으니 꽉 채우길 권장).

## 넣는 법
1. 위 이름으로 PNG 저장 → 이 폴더(`assets/maps/`)에 넣기
2. Godot 에디터를 한 번 켰다 끄면(임포트) 다음 실행부터 반영
3. 마음에 안 들면 파일만 지우면 자동 생성으로 복귀
